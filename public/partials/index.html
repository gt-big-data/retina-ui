What is RETINA?

REal-Time Intelligent News App

About

Under development by Georgia Tech's Big Data Club, RETINA compiles top news sources and conveys the information into a web/mobile interface complete with news recommendations.

Main Components of the Project

Crawling - Integrating all sources to links in HTML.
Clustering - Organizes data based on titles and keywords.
API - Returns news results based on recommendations
UI - Responsible for the web/mobile app and account creation/registration.
